# ToM-in-AMC #

Pointers to some codes and data used in *[Few-Shot Character Understanding in Movies as an Assessment to Meta-Learning of Theory-of-Mind](https://arxiv.org/pdf/2211.04684.pdf)*

## Data ##
Please find the preprocessed data at *[GoogleDrive/tom_in_amc](https://drive.google.com/drive/folders/19zrsc84pMCFJ4bXY4nORXMCGMN_nQFME)*
* **tom_in_amc.v1.0.json** -- movie scripts split into scenes with main characters masked
* **tom_in_amc.v1.0.bart_tok.json** -- the BPE tokenized data that can be directly used with our code
